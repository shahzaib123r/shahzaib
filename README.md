# 🤖 Shadow Bot — WhatsApp Automation

> Powerful Baileys based WhatsApp Bot ⚡

✨ Features: Anti-Delete 🗑️ | View Once Recovery 👁️ | Auto Status View 📺 | Auto React 💬 | Status Save 💾 | TagAll 🏷️ | Sticker Maker 🎭 | Stats & Logs 📊

⚙️ Setup:
npm install
node index.js

📋 Commands:
.help → Menu  
.alive → Bot Status  
.vv → View Once Recovery  
.sticker → Sticker Maker  
.tagall → Mention All Members  
.stats → Bot Stats (Owner)  
.logs → Deleted Messages (Owner)
